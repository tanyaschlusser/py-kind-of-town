SEC API

Start here:
sec.gov/developer

  • All the forms filed by public companies are accessible on the public EDGAR API
  • EDGAR (Electronic Data Gathering, Analysis, and Retrieval system)
    sec.gov/edgar/searchedgar/accessing-edgar-data.htm

EDGAR data

(no APIkey)
  • max 10 queries/sec (pretty please); else possibly block your IP
  • All SEC filings since 1994
  • HTML, JSON, and XML
  • need some regex to pull out the parts you want

Paid data

(yes APIkey)
  • EDGAR¬ÆOnline DataFied API:
    • max 2 queries/sec (docs)
    • Need API key: (developer.edgar-online.com/member/register)
    • JSON (default) and XML
